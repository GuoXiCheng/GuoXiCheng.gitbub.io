## Nest CLI

```bash
# 创建 controller
nest generate controller <name>

# 创建 service
nest generate service <name>

# 创建不带 spec 文件的 controller
nest generate controller <name> --no-spec

# 模拟创建 controller
nest generate controller <name> --dry-run
```
