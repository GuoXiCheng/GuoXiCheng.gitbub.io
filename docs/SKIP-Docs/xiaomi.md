# 小米手机使用

## 应用后台锁定
进入「后台管理」，长按「SKIP」，锁定

<figure markdown>
![Image title](https://pic.imgdb.cn/item/64a2ce701ddac507cc578a72.jpg){ width="300" }
</figure>

## 忽略电池优化
进入「应用信息」，进入「省电策略」，选择「无限制」

<figure markdown>
![Image title](https://pic.imgdb.cn/item/64a2ce711ddac507cc578b2a.jpg){ width="300" }
</figure>

## 允许自启动
进入「应用信息」，打开「自启动」
